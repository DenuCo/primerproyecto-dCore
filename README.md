# dCore 📖✨

Un espacio digital diseñado para compartir ideas originales, enfocado en la difusión de cómics y e-books.

## Descripción
Este proyecto es una página web desarrollada como proyecto práctico para consolidar los primeros pasos en el desarrollo Front-End dentro del programa Talento Tech. 

El objetivo principal de dCore es transformarse en una plataforma comunitaria. La propuesta actual es sencilla: crear una interfaz para subir y descubrir cómics, e-books y, más adelante, audiolibros, compartiéndolos con una audiencia.

## Tecnologías y Herramientas Utilizadas
El proyecto está construido aplicando fundamentos de desarrollo web:
* **HTML5:** Estructura semántica clara (utilizando `<header>`, `<main>`, `<nav>`, `<footer>` y `<section>`) para organizar el contenido y mejorar la accesibilidad.
* **CSS3:** Maquetado general de la aplicación combinando **CSS Grid** (para el layout principal) y **Flexbox** (para menús y grillas de contenido).
* **Tipografías e Íconos:** Integración de Google Fonts (*Indie Flower* y *Winky Sans*) y FontAwesome.
* **Formularios:** Integración con Formspree para el envío real de mensajes desde la sección de contacto.

## Estructura Actual
* **Hero / Tendencias:** Sección destacada para la obra principal del momento.
* **Galerías:** Visualización en formato de tarjetas (*cards*) para "Nuevos lanzamientos", "Cómics Populares" y "E-Books Populares".
* **Navegación:** Menú lateral interactivo.

## Roadmap (Próximas Versiones)
dCore está pensado para seguir creciendo. En futuras actualizaciones se implementará:
- [ ] **Publicación de Usuarios:** Los lectores también podrán optar por subir sus propias obras.
- [ ] **Filtro de Inteligencia Artificial:** Las obras subidas por los usuarios pasarán por una evaluación mediante IA. Este sistema buscará evitar plagios y dará recomendaciones de modificación, o bien dará "luz verde" para su publicación en la página.
- [ ] **Audiolibros:** Nueva sección dedicada a formatos de audio.
